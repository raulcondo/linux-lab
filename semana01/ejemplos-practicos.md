# Ejemplos Prácticos de Navegacion
---

## /home

Ejemplo: Cuando un usuario guarda un documento, descarga una imagen o crea un archivo de texto, estos se almacenan dentro de su carpeta personal en /home.

---

## /etc

Ejemplo: Cuando un administrador configura la red o cambia parámetros del sistema, los archivos de configuración se guardan en este directorio.

---

## /var

Ejemplo: Cuando un servidor web está funcionando, los registros de acceso y errores se almacenan en /var/log.

---

## /tmp

Ejemplo: Cuando se instala un programa o se ejecuta una aplicación, se crean archivos temporales en este directorio que luego se eliminan automáticamente.

---

## /usr

Ejemplo: Cuando se instala un programa como un editor de texto o un navegador, sus archivos principales se guardan en este directorio.

---

## /bin

Ejemplo: Cuando el usuario ejecuta comandos básicos como copiar archivos o listar carpetas, los programas utilizados se encuentran en este directorio.

---

## /boot

Ejemplo: Cuando el sistema se enciende, Linux utiliza los archivos almacenados en este directorio para iniciar correctamente.

---

## /dev

Ejemplo: Cuando se conecta una memoria USB, el sistema la representa como un archivo dentro de este directorio.

---

## /proc

Ejemplo: Cuando se desea verificar información del procesador o de la memoria, el sistema proporciona esos datos mediante archivos virtuales en este directorio.
